# Nathan Rasmussen Design

Static portfolio website for Nathan Rasmussen Design, built with Astro and intended for GitHub Pages at `nathanrasmussen.com`.

## Commands

```sh
npm install
npm run dev
npm run build
npm run preview
```

## Updating Content

- Site-wide settings live in `src/data/site.ts`.
- Portfolio projects live in `src/data/projects.ts`.
- Placeholder images live in `public/images/` and can be replaced with real project images while keeping the same filenames or by updating the `image` paths in project data.
